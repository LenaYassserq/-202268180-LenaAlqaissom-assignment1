## Lena Alqaissom – Portfolio

This is a single-page personal portfolio website developed for Lena, a junior Software Engineering student at King Fahd University of Petroleum and Minerals (KFUPM).The website was created as part of an academic assignment to demonstrate fundamental web development skills using HTML, CSS, and JavaScript, along with AI-assisted development tools.

---

### Features

- Responsive design that works on desktop, tablet, and mobile devices.
- About section highlighting academic background, technical skills, and professional aspirations
- Projects section showcasing selected academic work:
  - Database Systems (ICS 321) – SQL-based database project.
  - Student Daily Schedule Route Visualizer (SWE 316).
  - Artificial Intelligence Classification Project (COE 292).
- Contact section with a simple form (Name, Email, Message).
- Quick access links to LinkedIn and GitHub.
